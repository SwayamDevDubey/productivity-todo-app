# Commands to run after creating your GitHub repository
# Replace YOUR_GITHUB_USERNAME with your actual username

# Step 1: Add GitHub repository as remote
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/productivity-todo-app.git

# Step 2: Push your code to GitHub
git push -u origin main

# Example with SwayamDevDubey username:
# git remote add origin https://github.com/SwayamDevDubey/productivity-todo-app.git
# git push -u origin main

# After successful push, your app will be available at:
# https://github.com/YOUR_GITHUB_USERNAME/productivity-todo-app

# To enable GitHub Pages (free hosting):
# 1. Go to repository Settings
# 2. Scroll to Pages section  
# 3. Source: Deploy from branch
# 4. Branch: main
# 5. Save
# Your app will be live at: https://YOUR_GITHUB_USERNAME.github.io/productivity-todo-app
